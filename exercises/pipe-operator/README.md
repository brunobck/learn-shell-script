# Operador pipe `|`

Neste exercício iremos interagir com o gerenciamento de processos no Linux. Seu
objetivo é contar todos os processos atualmente ativos no sistema que ocupam
10MB ou mais de memória.

Dica: Chamando o `ps -aux` você conseguirá listar todos os processos que estão
rodando atualmente. A coluna `RSS` indica o uso de memória física (em
kilobytes), é a partir dessa coluna que você irá **filtrar** e **contar** os
processos.
